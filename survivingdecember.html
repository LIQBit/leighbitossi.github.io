Surviving December - Band Website

In May 2021, I was asked to build a site for a UK-based punk-rock band called Surviving December. The band had recently signed to a label and were looking to build an online presence. Initially, I built a simple static site using CSS, HTML and some JavaScript. Later, I built a headless CMS blog using Contentful for the site using Gatsby/React and redirected the site's DNS nameservers to Netlify. Initially, the band really liked the look of another band's website this site and they asked if I could create something very silmilar. The site makes use of various widgets such as Spotify, YouTube and bandsintown. here.
